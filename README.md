# GameZone - Your Video Game Hub

## 📋 Project Overview

**GameZone** is a responsive web application designed as your ultimate source for all things video games. This project showcases popular games, provides game reviews, and delivers the latest gaming news in an engaging and user-friendly interface.

## 🎮 Features

- **Navigation Menu**: Easy navigation to different sections of the site (Home, Popular Games, Reviews, News, Contact)
- **Popular Games Section**: Display of featured games with descriptions and "Learn More" buttons
- **Game Reviews**: Articles and in-depth reviews of popular games
- **Latest Gaming News**: Keep up with the latest developments in the gaming industry
- **Contact Form**: Allow users to reach out with questions and feedback
- **Responsive Design**: Clean and modern styling with CSS for a professional appearance

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)

### Installation

1. Clone or download the project files to your local machine
2. Navigate to the project directory
3. Open `video games.html` in your web browser

### Running the Application

Simply open the `video games.html` file in any web browser to view the GameZone website.

## 🎨 Design Features

- **Header**: Dark background with white text and centered navigation
- **Game Cards**: White cards with rounded corners and subtle shadows showcasing game information
- **Buttons**: Orange-themed buttons with hover effects for better user interaction
- **Typography**: Arial font family with clean, readable line spacing
- **Color Scheme**:
  - Primary Dark: `#222` (Header)
  - Accent Orange: `#ff9800` (Buttons and links)

## 🔧 Customization

To customize the site:

1. **Update Game Information**: Edit the game names, descriptions, and images in `video games.html`
2. **Adjust Colors**: Modify CSS color values in `video games.css`
3. **Add More Games**: Duplicate the game-card div and update with new content
4. **Replace Images**: Swap image files while maintaining the same filenames
